# Tennis Courts Academy

## Règles métier utilisées par le tutoriel

Version 1.0

------------------------------------------------------------------------

# Objectif

Ce document décrit les règles de jeu exploitées par le tutoriel.

Il ne décrit pas les dialogues mais les mécaniques que les modules
doivent enseigner.

------------------------------------------------------------------------

# Echange

Un match est constitué d'une succession d'échanges.

Chaque échange est indépendant.

À chaque échange :

-   les deux joueurs reçoivent leur main ;
-   chacun dispose de son endurance ;
-   un serveur engage le point.

------------------------------------------------------------------------

# Endurance

L'endurance est la ressource permettant de jouer des cartes.

Chaque carte possède un coût.

Lorsqu'un joueur ne souhaite plus ou ne peut plus jouer, il passe.

------------------------------------------------------------------------

# Puissance

Chaque carte rapporte des points de puissance.

Les puissances sont cumulées pendant l'échange.

En fin d'échange, elles sont comparées.

------------------------------------------------------------------------

# Passer

Passer met fin à l'échange.

Le joueur qui passe accorde à son adversaire un bonus égal à son
endurance restante.

Bonus minimum : 2 points.

------------------------------------------------------------------------

# Résolution

Comparer les puissances finales.

Le joueur ayant le plus de puissance remporte l'échange.

En cas d'égalité, le serveur remporte l'échange.

------------------------------------------------------------------------

# Boost

Le boost constitue une condition de victoire immédiate.

Lorsqu'un boost est joué :

-   l'adversaire doit répondre correctement ;
-   s'il passe ou échoue, il perd immédiatement l'échange.

Les points de puissance ne sont alors plus pris en compte.

------------------------------------------------------------------------

# Cartes Effet / Remise

Une carte Effet / Remise peut :

-   être jouée pour son effet ;
-   être utilisée en remise pour ajouter du placement.

Le tutoriel présentera progressivement ces deux usages.

------------------------------------------------------------------------

# Effets

Les effets sont résolus lorsque leurs conditions sont réunies.

Le tutoriel doit toujours expliquer un effet avant de demander au joueur
de l'utiliser.

------------------------------------------------------------------------

# Progression pédagogique

Module 1 : Bases du jeu.

Module 2 : Premier échange.

Module 3 : Fin d'échange.

Module 4 : Précision et placement.

Module 5 : Boost.

Module 6 : Défendre un boost.

Module 7 : Effet / Remise.

Module 8 : Couleurs de boost.

Module 9 : Effets avancés.

------------------------------------------------------------------------

# Règle générale

Une seule nouvelle mécanique est introduite par module.

Les modules suivants peuvent réutiliser les mécaniques précédentes mais
ne doivent pas introduire plusieurs concepts majeurs simultanément.
